# Y-FLEX MO  
**A Web-Based Gamified Yoga Instructional Module**

---

## ğŸ“˜ Project Overview

**Y-FLEX MO** is a web-based instructional module designed to support the teaching and learning of **yoga poses** through **gamified challenges**, **self-paced activities**, and **formative assessments**. The platform was developed as part of an academic research project in **Physical Education**, focusing on the integration of **technology-enhanced instruction** in movement-based learning.

The module is deployed using **GitHub Pages**, enabling open-access delivery of instructional materials without the need for server-side processing.

---

## ğŸ¯ Purpose of the Project

The primary purpose of this project is to:

- Develop a **web-based learning tool** for yoga instruction  
- Support **flexibility, balance, and pose accuracy** through guided challenges  
- Provide **interactive and engaging instructional content**  
- Serve as an **instructional intervention** for educational research  

This platform is intended for **pilot testing, validation, and instructional implementation**, not for long-term data storage or summative grading.

---

## ğŸ§  Instructional Design Framework

The design of Y-FLEX MO is grounded in established educational theories and models, including:

- **Multimedia Learning Theory** (Mayer, 2009)
- **E-Learning and Instructional Design Principles** (Clark & Mayer, 2016)
- **Formative Assessment Theory** (Black & Wiliam, 2009)
- **Online Learning Environments** (Anderson, 2008)

Gamification elements (e.g., challenges, rewards, progress tracking) are incorporated to enhance learner motivation and engagement.
